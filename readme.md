# Set up instructions
- Create a s3 bucket
- AWS_ACCESS_KEY_ID
- AWS_SECRET_ACCESS_KEY
- S3_REGION
- AWS_BUCKET

# Development
- npm run dev

# Extra information
- USER uploads file to s3 and gets storred in s3
- Clinet will get the stored url and sign it
- Signed url will then be loaded as csv and rendered on client
